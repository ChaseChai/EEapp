# MathBlaster (数学爆破者) - 东方墨韵数学交互游戏

👉 **[点击这里直接在线游玩 / Play Now](https://ChaseChai.github.io/MathBlaster)**

MathBlaster 是一款结合了“东方现代主义（吴冠中彩墨风格）”美学与高等数学逻辑的交互式解谜网页游戏。在这里，抽象的数学函数将化身为宣纸上的水墨轨迹，玩家需要通过寻找并调试数学公式，指引墨滴精准命中所有目标。

## 🎮 游玩方法 (How to Play)

### 1. 观察目标点与提示
进入每个关卡后，坐标系中会出现若干个**朱砂红色的目标点**。屏幕上方会提供该关卡的**通关提示 (Hint)**，暗示你需要构造哪种类型的数学函数（如直线、抛物线、正弦波、指数衰减等）。

### 2. 编写与调整数学公式
- **公式框架与滑块**：游戏下方提供了带变量的数学公式（例如 `a * x^2 + c`）。底部有 `a`, `b`, `c` 三个参数滑块。
- **动态调整**：您可以拖动滑块来实时改变函数曲线的形态（如振幅、开口大小、相位、偏移量等）。
- **轨迹预览**：在调整滑块时，宣纸画布上会实时显示一条**虚线轨迹**，代表当前公式的飞行路线。

### 3. 发射墨滴 (Launch)
- 当你认为虚线轨迹已经完美穿过所有目标点时，点击页面底部的 **"Launch (发射)"** 按钮。
- 墨滴将顺着你构建的函数轨迹高速飞行。若成功触发所有目标点，即可通关并进入下一关卡！

## 🎨 视觉与美学 (Design Philosophy)

游戏的视觉语言融合了中国水墨画的留白与现代抽象几何：
- **背景空间**：宣纸白 (`#fbfbf9`) —— 提供可呼吸的负空间（留白）。
- **主线笔墨**：焦墨 (`#1c1c1c`) 与 淡墨 (`#5e5e5e`)。
- **点缀色调**：朱砂红 (`#d44d4d`)、石绿 (`#5a7d65`)、藤黄 (`#d1b46b`)、花青 (`#4f8a8b`)。

## 🌌 关卡概览 (Level Guide)

游戏包含 9 个精心设计的关卡，难度循序渐进，极具挑战性：

1. **Origin (新手)**：一条简单的直线，感受 `y = a * x` 的斜率变化。
2. **Curve (中等)**：基础二次函数抛物线，像一个温柔的微笑。
3. **Gravity (中等)**：模拟真实的重力抛物线轨迹，向下坠落。
4. **Offset (困难)**：不仅有弧度，还发生了水平与垂直平移，需要结合 `b` 和 `c` 进行偏移调整。
5. **Construction (困难)**：无尽的振荡波纹（基于 `sin` 正弦函数）。
6. **Resonance (进阶)**：抛物线与直线的叠加态（共振）。
7. **Damping (进阶)**：指数衰减（`exp`），能量在原点聚集后像钟声一样向外消散。
8. **Pulse / Agnesi (微积分)**：阿涅西的女巫曲线，脉冲般瞬间爆发又迅速回归平静的理性之美。
9. **The Abyss / Sigmoid (微积分)**：从虚无到一切的平滑过渡（逻辑斯蒂 S 型曲线）。

## 💻 技术栈 (Tech Stack)

- **框架**: Next.js (App Router) + React
- **样式**: TailwindCSS (自定义配色体系)
- **渲染**: HTML5 Canvas + `requestAnimationFrame` (60fps 高性能物理碰撞与粒子系统)
- **数学引擎**: `math.js` (安全的动态表达式解析)
- **部署**: GitHub Pages
