# MathBlaster - Eastern Modernism

MathBlaster is an interactive mathematical art project built with Next.js, React, and TailwindCSS. It transforms abstract mathematical functions into visual geometric trajectories, inspired by the "Eastern Modernism" aesthetic and the stroke techniques of Wu Guanzhong (Œ‚π⁄÷–).

## Design Philosophy 

The visual language follows the principles of Chinese ink painting mixed with modern abstraction:
- **Background**: Rice Paper White (`#fbfbf9`) - providing a breathable negative space
- **Primary Ink**: Dark Ink (`#1c1c1c`) & Light Ink (`#5e5e5e`)
- **Accent Colors**: Cinnabar Red (`#d44d4d`), Mineral Green (`#5a7d65`), Rattan Yellow (`#d1b46b`), Cyan (`#4f8a8b`)

## Tech Stack
- Next.js (App Router)
- React + TailwindCSS
- Framer Motion & HTML5 Canvas
- math.js for safe mathematical evaluations
