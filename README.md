# 🎮 MathBlaster —— 数学函数弹道游戏

## 一、项目概述

**MathBlaster** 是一个基于数学函数轨迹的二维交互式小游戏。
玩家通过输入数学函数，控制小球的运动轨迹，使其经过关卡中的多个目标点，从而完成挑战。

游戏将抽象的数学函数转化为直观的动态弹道，让玩家在“构造函数”和“观察轨迹”的过程中理解数学图像与参数变化。

---

## 二、核心设计理念

传统数学学习中：
* 函数往往只是公式
* 图像与运动缺乏直观联系
* 学习过程偏静态

而 MathBlaster 的核心思想是：
> **把函数从“纸面表达”变成“可交互轨迹”**

玩家不再只是“计算函数”，而是：
* 观察函数
* 构造函数
* 调整函数
* 用函数解决问题

从而形成一种轻量级、游戏化的数学体验。

---

## 三、核心玩法

### 玩家目标
通过输入合适的函数（例如：`y = ax² + bx + c`），让小球沿函数轨迹飞行，并命中关卡中的所有目标点。

---

## 四、游戏流程

### 1️⃣ 开始界面
显示游戏标题、操作提示与简介。

### 2️⃣ 进入关卡
每一关都会：
* 显示多个目标点
* 提示当前关卡目标
* 提供函数输入框

### 3️⃣ 玩家输入函数
例如输入：`-0.2*x*x + 4*x`。系统会自动解析函数、转换数学表达式并准备生成对应轨迹。

### 4️⃣ 发射阶段
按 Enter 后，小球从左侧发射，沿函数轨迹逐点运动，实时检测是否命中目标。

### 5️⃣ 命中反馈
命中目标后，目标发生状态变化，过关得分增加。

### 6️⃣ 通关 / 失败
* **全部命中**：提示成功，进入下一关。
* **未全部命中**：重置球的位置，玩家重新调整函数并尝试。

## 五、关卡系统设计（核心亮点）

游戏采用固定关卡设计，而非随机生成。

### 设计原则
每一关都对应一个“隐藏的过关函数逻辑”，所有目标点都基于该函数分布，玩家需要“反推函数轨迹”。因此，每一关本质上都是一个“函数构造问题”。

## 七、技术实现方案

本项目采用轻量化的 Web 端结构开发完成，可以直接部署在 GitHub Pages 供在线访问：

### 技术栈
* **核心渲染与逻辑引擎**：JavaScript + `p5.js`（核心动画渲染、数学转换与交互）
* **运行层载体**：HTML5 / CSS3（响应式布局支撑）
* *(保留项：在 `python_prototype/` 目录中封存了初始基于 Python + pygame 的核心架构验证版)*

### 核心模块
1. **函数解析**: 解析玩家输入的合法化，拦截并替换表达式里的各种运算符（如把特定自然常数或者符号平滑化处理成标准 JS 表述）。
2. **轨迹生成**: 通过计算 `y = f(x)` ，基于特定步长（`step`）实时采样点生成的平滑轨迹。
3. **坐标变换**: 数学坐标转屏幕像素映射模块驱动，构建屏幕世界到笛卡尔直角坐标坐标的高精度等比系转换。
4. **动画系统**: 结合 p5 刷新循环 (`draw`) 推动弹道动画渲染。
5. **碰撞检测引擎**: 通过两点间距离函数进行基于点位的实时圆心距测算触发得分判定。
6. **关卡管理系统**: 设计精简而完善的独立有限游玩状态机逻辑跳转 (过关/失败/动画态等)。

## 八、项目亮点

* ✅ **将数学“动态化”**：函数不再是静态公式，而是真实运动轨迹。
* ✅ **游戏化学习**：把函数学习转化为挑战和探索。
* ✅ **可视化直观**：帮助理解函数参数组合与图像曲率间的变化关系。
* ✅ **交互感强**：玩家自己充当了底层缔造者，主动来试验这个物理世界。
* ✅ **AI 前沿开发模式**：项目架构设计与迁移全程使用了 Vibe coding + AI 驱动进行高效原型迭代构建。

## 九、未来可扩展方向

1. **参数控制滑块**: UI 层面上提供控制台控件拖拽调节各项常参数 $a, b, c$ 等。
2. **增强数学库支持**: 加入分段/极坐标/抽象对数函数的深渊挑战。
3. **编辑器沙盒体系**: 上线沙盒功能支持游戏 UGC内容发布交流，互相做谜语与出题。
4. **竞技排行系统与重播**: 设立最短解法字符挑战、极速解题等激励打分反馈榜单。
5. **智能数学 AI 弱提示**: 通过轨迹推演与失败位置给玩家做一定的“调整抛物线系数开口度”的大语言教辅建议。

## 十、项目总结

> **MathBlaster** 尝试将高度抽象化与纸面化的数学函数计算，转化为可交互、极具感官及控制正向循环反馈逻辑的即玩式游戏操作体验。玩家透过把控多维函数底层模型掌控轨迹生成，在游玩过程中不再停留于被填鸭死记知识，而是通过一次次“解开方程式谜底与可视化校正”对深藏函数内的微调敏感度有了最直观领悟。作为将“生硬推演”转换为“学游并重与现代美学探索”的技术探索实践品，项目的 Web 在线零配置轻便游玩特性更利于后续做校园/社交化的扩散和交互教育产品迭代示范！
